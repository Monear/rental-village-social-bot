# Examples Directory

This folder can contain real examples of AI-generated content (optional).

## Structure (If Adding Examples)

```
examples/generated-content/
└── campaign-name/
    ├── post-example.md          # AI-generated text
    ├── post-example.png          # Image used
    └── README.md                 # Campaign summary with metrics
```

## Example Campaign Summary Template

If you add examples, include a README.md with:

```markdown
# Campaign Name

**Generated:** Date
**Posts Created:** Number
**Generation Time:** Time with AI
**Review Time:** Time for human polish

## Results
- Engagement metrics
- Business impact
- What worked well

## Example Posts
- Link to individual post files
```
