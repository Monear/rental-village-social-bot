# Rental Village Social Bot

This repository contains the Python scripts and strategy documents for the Rental Village social media automation project. The system uses AI to suggest content ideas, which are then approved and produced by a human team. Automation handles the posting of content and the tracking of performance metrics.

## Core Components

-   **/src**: Contains all Python scripts for automation.
-   **/strategy_documents**: Contains all strategic planning and branding documents.
-   **/docs**: Contains implementation and usage guides.
-   **Dockerfile**: Defines the containerized environment for running the scripts.
-   **.env**: Stores local environment variables (should not be committed).
